Installation
============

You can install the latest release of exceltopostgresql at `PyPI`_ via `pip`_:

.. code-block:: python

    pip install exceltopostgresql

By running the command above, the exceltopostgresql package along with required dependency packages
(`sqlalchemy`_, `pandas`_) will be installed to your computer (if they have not been installed yet).


.. _`PyPI`: https://pypi.org/project/exceltopostgresql
.. _`pip`: https://packaging.python.org/key_projects/#pip
.. _`sqlalchemy`: https://github.com/sqlalchemy/sqlalchemy
.. _`pandas`: https://github.com/pandas-dev/pandas
.. _`our repository`: https://github.com/xyluo25/exceltopostgresql
