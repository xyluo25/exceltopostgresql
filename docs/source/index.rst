exceltopostgresql
====================================
| **Authors**: Xiangyong Luo
| **Email**: luoxiangyong01@gmail.com


exceltopostgresql is a tool to convert excel files (.xls, .xlsx, .csv) to mysql database.

**If you use exceltopostgresql in your research, please cite: https://github.com/xyluo25/exceltopostgresql**


**Package dependencies**
------------------------

* [X] sqlalchemy
* [X] pandas
* [X] pyscopg2
* [X] pyufunc
